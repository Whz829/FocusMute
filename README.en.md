# FocusMute

FocusMute is a Windows desktop app that automatically mutes background processes and restores their audio when they return to the foreground.

## Features

- Automatically detect the current foreground window
- Mute background audio processes
- Restore audio automatically when an app returns to the foreground
- Support both whitelist and blacklist modes
- Support launch at startup
- Support minimize to tray
- Show muted processes and activity logs in real time
- Extract and display process icons
- Overview status cards
- Running process table
- Per-process audio status, foreground status, and policy status
- Whitelist and blacklist management
- Toggles for auto mute, startup launch, and minimize to tray
- Light and dark themes
- Real-time log panel
